新增文章
```bash
$bundle exec rake new_post['post_title']
```
編輯文章
```
$vim source/_posts/post_file
```
預覽網站
```bash
$bundle exec rake preview
```
生成網站
```bash
$bundle exec rake generate
```
發佈網站
```bash
$bundle exec rake deploy
```
